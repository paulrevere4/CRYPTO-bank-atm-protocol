# CRYPTO-bank-atm-protocol
Bank-Atm protocol for Cryptography and Network Security I

Depends on `g++` and `libcrypto++-dev`, to install dependencies on a clean install of Ubuntu run:
```
$ source setup.sh
```

To build, run:
```
$ source build.sh
```

Usage:
```
$ ./atm <port1>
$ ./bank <port2>
$ ./proxy <port1> <port2>
```
